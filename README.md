

#  Price Tracker Chrome Extension

A Chrome extension that automatically tracks product prices and helps users monitor price changes over time.  
It is designed to simplify online shopping by keeping track of price fluctuations and enabling smarter buying decisions.

---

## 🚀 Features

- 🔍 Track product prices from web pages
- 📉 Monitor price changes over time
- ⏰ Automated price tracking (daily scheduler support)
- 💾 Store product and price data locally
- ⚡ Lightweight and easy to use

---

## 🧠 How It Works

1. The extension captures product details from supported websites  
2. Price data is stored and updated regularly  
3. A Python-based script can run daily to track price changes  
4. Users can view and analyze price trends  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Extension API:** Chrome Extensions API  
- **Backend Logic:** Python  
- **Data Handling:** CSV / Local storage

## ⚙️ Installation Guide

### 🔹 Load Extension in Chrome

1. Download or clone this repository  
2. Open Chrome and go to:
3. Enable **Developer Mode** (top right)  
4. Click **Load unpacked**  
5. Select the project folder  

---

## ▶️ Usage

- Open any product page  
- Use the extension to capture product details  
- Run the Python script for tracking (if applicable)  
- Monitor price changes over time
